# Quiltt Nuxt Example

This Nuxt app demonstrates how to integrate `@quiltt/vue` and provides Playwright e2e coverage for Vue SDK behavior.

## Getting Started

```bash
pnpm install
pnpm run dev
```

Open <http://localhost:3000>.

## E2E Tests

```bash
pnpm run playwright:install
pnpm run test
```
